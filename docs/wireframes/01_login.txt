FLOW — HAPPY PATH:
  1. User lands on /login
  2. Clicks "Login with Kite"
  3. Browser redirects to Zerodha OAuth page
  4. User enters Zerodha credentials + TOTP on Zerodha
  5. Zerodha redirects to /auth/callback?request_token=...
  6. Backend exchanges token, issues JWT cookie
  7. Browser redirects to /dashboard
